# CSC-125-201 Final Project
# Project Description
The goal of the program is to help students calculate their overall GPA using grades from multiple classes. After all classes are entered, the program converts the numeric grades into letter grades and then calculates the GPA
# Features
- Dictionary storing class information (Final Grade, credits). The classname is the key, and the value stores the numeric grade and the number of credits for that class.
- Custom function `convert_to_letter_grade` that This function takes a numeric grade as a parameter and returns the corresponding letter grade.
- [for or while] loop that loop over inputs allowing users to enter multiple classes. The loop keeps running until the
user types "done"
- User input with validation (Using try/except + While loop)
- Formatted output
